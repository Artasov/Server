Никита Артасов

    Начал заниматься программированием с 2020 года. Изначально осваивал desktop разработку на основе python Qt framework. Через несколько месяцев изучения попробовал QT C++, увидев некоторые преимущества (компиляция в executable файл например), начал изучать его. Работа с WinAPI, парсинг данных с различных api, статическая сборка проекта, работа с потоками, отправкой данных на сервер и т.п. В университете учил Си, ничего "крупного" на нем не писал. После создания нескольких, относительно крупных, приложений на QT, начал изучать front-end dev. Html, css, js, sass. Поверхностно смотрел bootstrap и webpack. После того как сверстал первый, по моему мнению, нормальный, адаптивный, стилизованный сайт, захотелось изучать backend. Я выбрал Django потому, что это простой для понимания, как мне показалось, framework на знакомом мне языке, с читабельной документацией. Тут уже началась работа с MTV моделью, с SQLite3 после с MySql, PostgreSQL, шаблонизаторами, Celery, Redis, настройкой админ панели и т.д. Умею пользоваться DRF. Курсов не проходил. Всё учил по видео с youtube, а после быстрого наступления их нехватки для решения возникающих проблем, углублялся в документацию или искал что-то в англоязычном сегменте youtube. Пример разработки - https://xlartas.ru/, адаптивный сайт развернутый на timeweb с mysql, с регистрацией пользователей, с мини-магазином моих же приложений для компьютерной игры, с подтверждением по почте, с recaptcha, с оплатой товара через Qiwi api. Такие инструменты как vue.js, node.js, typescript, react, не изучал, но планирую постепенно учить. Опыта коммерческой разработки не имею, но очень бы хотелось получить. Готов изучать неизвестные технологии.
